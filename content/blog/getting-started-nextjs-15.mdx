# Getting Started with Next.js 15: A Complete Guide

Next.js 15 brings significant improvements to the already powerful React framework. In this guide, we'll explore the key features and how to get started.

## What's New in Next.js 15?

- **Improved Performance**: Faster builds and optimized runtime
- **App Router Enhancements**: Better routing capabilities
- **Server Components**: Enhanced server-side rendering
- **TypeScript Support**: First-class TypeScript integration

## Installation

```bash
npx create-next-app@latest my-app
cd my-app
npm run dev
```

## Project Structure

```
my-app/
├── src/

│   │   ├── page.tsx
│   │   └── layout.tsx
│   └── components/
```
