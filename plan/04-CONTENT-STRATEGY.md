## üéØ SEO Strategy

### On-Page SEO Checklist
```tsx
// Every blog post must have:
<Head>
  <title>Primary Keyword | Kunal Vishwakarma</title>
  <meta name="description" content="150-char compelling summary with keyword" />
  <meta name="keywords" content="primary, secondary, tertiary" />
  
  {/* OpenGraph */}
  <meta property="og:title" content="..." />
  <meta property="og:description" content="..." />
  <meta property="og:image" content="/og-image.jpg" />
  <meta property="og:type" content="article" />
  
  {/* Twitter */}
  <meta name="twitter:card" content="summary_large_image" />
  <meta name="twitter:title" content="..." />
  <meta name="twitter:description" content="..." />
  <meta name="twitter:image" content="/twitter-image.jpg" />
  
  {/* Schema.org */}
  <script type="application/ld+json">
    {JSON.stringify({
      "@context": "https://schema.org",
      "@type": "BlogPosting",
      "headline": "...",
      "author": {
        "@type": "Person",
        "name": "Kunal Vishwakarma"
      },
      "datePublished": "2024-12-01",
      "image": "/article-image.jpg"
    })}
  </script>
</Head>
```

### Article Structure (SEO-Optimized):
```markdown
# H1: Primary Keyword (Only One)

[150-word intro with keyword in first 100 words]

## H2: Related Secondary Keyword
[Content]

### H3: Supporting Point
[Content]

## H2: Another Secondary Keyword
[Content with internal links]

## H2: Common Questions (Target Featured Snippets)
### What is [keyword]?
### How to [keyword]?
### Why [keyword] matters?

## H2: Conclusion
[Summary with CTA]
```

### Technical SEO:
- **Sitemap**: Auto-generated with next-sitemap
- **Robots.txt**: Allow all, link to sitemap
- **Schema Markup**: Article, Person, Organization
- **Image Optimization**: WebP, lazy loading, alt text
- **Core Web Vitals**: LCP < 2.5s, FID < 100ms, CLS < 0.1
- **Mobile-First**: Responsive, touch-friendly
- **HTTPS**: SSL certificate required
